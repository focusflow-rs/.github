### About

FocusFlow is an ecosystem of productivity applications designed around one mission: helping people protect attention and finish meaningful work.

### Projects

<div align="center">

| Project | Description | Platform |
|:---:|:---|:---:|
| Focus Flow | Minimalist Pomodoro app with task context, settings, and productivity tracking | Web (WASM) |
| FocusFlow Desktop | Tray-first Pomodoro timer with notifications, persistent config, and session stats | Linux Desktop |

</div>
